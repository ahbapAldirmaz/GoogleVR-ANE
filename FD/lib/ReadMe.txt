This ANE is dependent on the following ANEs. You need to add these to your project too. this will help you avoid conflict error messages when using many different ANEs in your project.

gvr_common.ane
gvr_commonwidget.ane
gvr_panowidget.ane
gvr_videowidget.ane
androidSupport.ane
overrideAir.ane (required on the iOS side also)

Make sure you are using the latest version of these ANEs.
